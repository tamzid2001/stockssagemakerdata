Forecast

Generate quantile bands for the ticker in your chart and save the run so you can re-plot it later.

Daily = trading days. Hourly = hours.
Values must be between 0 and 1 (exclusive). Example: 0.1,0.25,0.5,0.75,0.9
Learn more

Forecasts are stored to your workspace as quantile bands. Quantiles are probabilities (not targets): for example, 0.1 is a lower band, 0.5 is the median path, and 0.9 is an upper band. Use them to reason about uncertainty, not to force a single outcome.

Forecast length controls how many future timestamps are generated for the selected timeframe. Daily forecasts use trading days (business days). Hourly forecasts are capped to keep the terminal responsive.

Meta Prophet is best when you want a transparent, classical model and stable uncertainty bands. IBM TimeMixer is best when you want a fast, foundation-model style forecast. Quantura saves both so you can compare them on the same chart.

After you generate a run, use Load saved forecast (or the Saved forecasts panel) to re-plot it later and export CSV for backtests.

Terminal basics

1. Load a ticker in the dock to render the chart.

2. Switch panels on the left. Your ticker context is reused so you can move fast.

3. Click any ticker pill (Trending, Screener, Watchlist) to update the chart immediately.

Forecasting

Use comma-delimited quantiles like 0.1,0.5,0.9. Quantura stores each run so you can re-plot it later. For analysis, export CSV and compare model bands against subsequent realized prices.

When you load a saved run, Quantura overlays the widest band (low-to-high quantile) plus an inner band (if available), then plots the median quantile as a dotted line.

Indicators

Indicators return two outputs: a table of latest values, plus a chart of full series. Trend overlays (SMA/EMA/Bollinger) are plotted on the main price chart so you can keep context.

Start with RSI and MACD. Add volatility context (ATR, Bollinger) before you interpret breakouts.

News, Options, and Discovery

Use News to validate catalysts and Options to inspect implied positioning. Use Trending and Screener to find candidates, then move back to Forecasting for uncertainty bands.

If you are building a research loop, store runs to your workspace and share it with collaborators from the Dashboard.
